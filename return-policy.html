Return Policy

Last updated: July 13, 2026

1. Return Guidelines

Product returns must be authorized in your dashboard to generate an RMA (Return Merchandise Authorization) label. Return shipments without labels are not recognized.

2. Product Packaging Requirements

All original accessories, manuals, adapters, and chargers must be packed inside the returning shipment boxes. Missing box elements result in return denials.
